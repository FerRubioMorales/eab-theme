# EAB Theme

Tema inspirado en la paleta natural de El Armario del Bebé (tonos lino, madera clara y verde botánico). Incluye compatibilidad con Elementor para maquetar cabeceras, pies y páginas, y soporte completo para WooCommerce (catálogo, carrito, galería de productos y lightbox) listo para flujos de drop shipping.

## Instalación
1. Copia esta carpeta en `wp-content/themes/eab-theme` de tu instalación de WordPress.
2. Activa **EAB Theme** desde el panel de administración en *Apariencia → Temas*.
3. Asigna tus menús a las ubicaciones **Primary**, **Footer** o **Shop**.
4. Añade widgets a la barra lateral desde *Apariencia → Widgets* (también se mostrarán en tienda).
5. Instala WooCommerce y tu plugin de drop shipping preferido; el tema ya incluye estilos para productos, botones y paginación.
6. Si usas Elementor, crea plantillas de **Header**, **Footer** o **Single** y el tema las respetará automáticamente.

## Desarrollo rápido
- Los estilos principales se encuentran en `style.css` (paleta: lino #fff9f1, arena #f1e4d4, verde #7b9f7b, terracota #c38b5f).
- El script para el menú móvil está en `assets/js/theme.js`.
- Plantillas clave: `index.php`, `single.php`, `page.php`, `woocommerce.php`, `header.php` y `footer.php`.

Puedes personalizar colores, tipografías o diseño editando estos archivos directamente o con un child theme.
